HTML code used from other websites and how we altered it:

    To make the cards on the "Home" and "Week's Journal" pages, we referenced w3schools.org for an idea on how to fit the image properly in the card. This code was altered on the homepage to not include any text and just a picture and was given a greyscale hover effect. The code on the "Week's Journal page was altered to make the cards horizontal and the images to alternately float left and right. All responsive design for the cards was written by Mariam.

    The code for the autoscroll on the "Week's Journal" page was taken from Design Shack. The code for the autoscroll itself was not altered, but we applied it the links on this page and formatted the links ourselves.

    The code for the hamburger style menu on resized windows and mobile devices was taken from w3schools.org. It was altered to to not include the home page link in the navigation bar and have the links drop down in the center. The size of the hamburger icon was increased, colors were changed, hover effects were changed, and margins were added to the menu.

    All other code for this website was written by Mariam.
